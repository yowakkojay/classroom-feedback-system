  已完成的功能

  后端 (Django)
  - 用户系统：三种角色（学生/教师/管理员），JWT密码登录，管理员CRUD

  前端 (Vue 3)
  - 登录页：用户名密码登录，按角色自动跳转
  - 学生端：查看活跃课堂、选择表情提交、进入讨论区

  启动方式

  # 后端
  cd backend && source venv/bin/activate && python manage.py runserver

  # 前端 (新终端)
